# reeln dock

**Cross-platform desktop companion for [reeln](https://github.com/StreamnDad/reeln-cli).**

Visual render profiles, clip review, and game management — all without touching the terminal. Built with [Tauri](https://tauri.app) and [Svelte](https://svelte.dev). Built by [Streamn Dad](https://streamn.dad).

> Coming soon. Follow this repo or visit [streamn.dad](https://streamn.dad) for updates.
